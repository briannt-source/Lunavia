## 🎯 Branding Guidelines

### Logo Usage
- **Full Logo**: Headers, landing pages, marketing materials
- **Icon Only**: Favicon, small spaces, mobile
- **Variants**: 
  - Light (white text) - trên dark backgrounds
  - Dark (dark text) - trên light backgrounds

### Color Usage
- **Primary Blue**: Buttons, links, active states, brand elements
- **Blue Dark**: Hover states, emphasis
- **Blue Light**: Backgrounds, subtle highlights
- **Slate/Gray**: Text, borders, secondary elements

### Typography
- **Font**: Inter (sans-serif)
- **Weights**: 400, 500, 600, 700
- **Sizes**: Responsive từ mobile đến desktop
